### 选择排序 (selection_sort)

**函数:**

```python
from salgorithm.sort import selection_sort

selection_sort(array)
```

**参数:**

* **array:** 要排序的数据列表

**返回值:**

* 返回排序后的数据列表

**示例:**

```python
array = [6, 4, 1, 8, 3, 9, 2, 5, 7]

sorted_array = selection_sort(array)

print(sorted_array)  # 输出: [1, 2, 3, 4, 5, 6, 7, 8, 9]
```
